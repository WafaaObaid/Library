# Library Management System (Java, JDBC, JavaFX)
## Overview

This project is an electronic Library Management System built using Java. It integrates JDBC for database connectivity and JavaFX with Scene Builder to create a modern, user-friendly graphical interface.

The system provides essential library operations, including managing books, members, and borrow/return transactions. All data operations are executed through structured JDBC queries to ensure efficient and reliable interaction with the database.

## Features

Add, update, delete, and search for books

Manage member information

Borrow and return book functionality

Interactive GUI designed using JavaFX and Scene Builder

Full database integration using JDBC

## Technologies Used

Java

JavaFX

Scene Builder

JDBC (MySQL / SQL Database)

## How to Run

Clone the repository

Import the project into your IDE (e.g., IntelliJ, Eclipse)

Connect the project to your SQL database

Run the JavaFX main application file
